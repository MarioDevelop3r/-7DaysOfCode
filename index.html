La tarea de hoy es reescribir el código:

            if (numeroUn == stringUn) {

                console.log('Las variables numeroUn y stringUn no tienen el mismo valor');

                //solución simplificada



                // constante que contiene los pares de números y strings
          const pares = [
            { numero: 1, string: "1" },
            { numero: 30, string: "30" },
            { numero: 10, string: "10" }
          ];

            // iterar sobre los pares y comparar los valores
          
          pares.forEach(({ numero, string }) => {
            console.log(
              numero == string
                ? `Las variables ${numero} y "${string}" tienen el mismo valor, pero tipos diferentes` 
                // si los valores son iguales pero los tipos son diferentes
                : `Las variables ${numero} y "${string}" no tienen el mismo valor`
                // si los valores no son iguales
            );

            // Comparar si los valores son iguales y del mismo tipo
         
            console.log( 
              numero === string
                ? `Las variables ${numero} y "${string}" tienen el mismo valor y el mismo tipo` 
                // si los valores y los tipos son iguales
                : `Las variables ${numero} y "${string}" no tienen el mismo tipo`
                // si los valores son iguales pero los tipos son diferentes
            );
          });

          // Fin de la solución simplificada

Explicación de la solución

  • En el primer y tercer bloque, se utiliza == para comparar el valor independientemente del tipo, lo que refleja que las variables tienen el mismo valor pero tipos diferentes.
  • En el segundo bloque, se utiliza === para comparar tanto el valor como el tipo, lo que refleja que las variables tienen el mismo valor y tipo.
